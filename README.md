# Sales Analysis - Power BI

## 📊 Project Overview
An interactive sales analysis dashboard built with Power BI, analyzing sales performance, customer behavior, and product trends across 2024-2026.

## 🛠️ Tools Used
- Power BI (Dashboard & Visualization)
- SQL (Data Modeling)
- Excel (Dataset & Budget Data)

## 📁 Project Structure
- `Dataset/` → Raw data files (Product, Customer, Calendar, Internet Sales, Budget)
- `SQLcode/` → Data model tables in PDF format
- `Visualization/` → Power BI dashboard file (.pbix) and PDF screenshots

## 📈 Key Insights
- Total sales of **22M+**, exceeding budget by **1M+**
- Bikes category dominates with **93.93%** of total sales
- Top customer: **Jordan Turner** with 15,999 in sales
- Top product: **Mountain-200** series leading all categories

## 🔧 Dashboard Features
- Sales vs Budget tracking
- Top 10 customers and products ranking
- Monthly and regional sales trend analysis
- Product category breakdown
- Interactive filters by year, month, city, and product
